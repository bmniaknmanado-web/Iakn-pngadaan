# Iakn-pngadaan
# Sistem Pengadaan dan Pemeliharaan IAKN Manado

Platform terpadu untuk manajemen pengadaan barang dan pemeliharaan fasilitas 
Institut Agama Kristen Negeri Manado.

## ✨ Fitur Utama

### Untuk Pengguna Biasa:
- 📊 Dashboard dengan statistik pengadaan dan pemeliharaan
- 📋 Menu pengadaan (ajukan, monitoring, cetak laporan)
- 🔧 Menu pemeliharaan (ajukan, monitoring, cetak laporan)
- 📥 Upload memo untuk pengajuan
- 📈 Progress tracking dengan timeline visual
- 📄 Export laporan ke PDF dan Excel

### Untuk Admin:
- 🔐 Login admin (username: admin, password: admin123)
- 📊 Admin dashboard dengan statistik lengkap
- 📝 Kelola data paket pengadaan/pemeliharaan
  - Pilih jenis: Barang, Pekerjaan Konstruksi, Jasa Konsultansi, Jasa Lainnya
  - Input nilai paket dalam Rupiah
  - Tambah keterangan paket
- 📤 Upload dokumen (BAST, Kuitansi, Laporan)
- 🗂️ Integrasi Google Drive (auto-create folder untuk setiap paket)
- 🔄 Kelola progress status paket (Memo → Pejabat → PPK → Selesai)

## 🚀 Teknologi

- Vite
- Tailwind CSS
- Lucide React Icons
- Vercel Hosting
